O script!

Agora vamos criar as variáveis que darão o suporte no Vue, essas serão criadas no
arquivo ou tag script.

const Home = {
template: `<div> Home Page <div/>`,
}
const About = {
template: `<div> About Page <div/>`
}
//lista (array) com objetos que tem a configuração da rota, tem 2 parametros
O path com o endereço e o component com o template

const routes = [
{ path: '/home', component: Home },
{ path: '/about', component: About }

]
//iniciar o roteador criando uma variavel para armazenar o roteador
const router = new VueRouter({ routes })

//Agora passamos o objte router para o Vue, antes dos 'data'

var app3 = new Vue({
el: '#app03',
router,

},
